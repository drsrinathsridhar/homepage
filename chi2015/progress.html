Author-wise Statistics

Pie charts of the number of lines edited by each author, and the number of commits per author. This data comes from the Subversion repository for the paper.

Note: These charts might not reflect reality due to two reasons.

  • Author A could commit multiple times to the repo changing only 1 character per commit thus increasing his/her commit count.
  • If author A writes 100 lines and author B moves it to a new file, then author B becomes the owner. This would obviously increase his/her line count.
